# Medical English Shadowing App

医療英語専門のシャドウイング・発音評価アプリ

## 技術スタック
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Infrastructure: GCP (Cloud Run + Cloud SQL)

## 開発環境セットアップ
```bash
# Docker起動
docker-compose up -d

# フロントエンド: http://localhost:5173
# バックエンド: http://localhost:3000
# DB: localhost:5432
```
